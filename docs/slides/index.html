Voice Cloning & TTS

Clone your voice from a 3-second sample. Generate natural speech. Fully local on Apple Silicon.
Local-Only Privacy-First Apple Silicon No API Keys
Pipeline
1
record-voice.sh
Capture a 5-10s voice sample via mic. 16kHz mono PCM WAV.
2
Qwen3-TTS 1.7B
Voice cloning model encodes your vocal identity from reference audio.
3
tts-speak.sh
Generate speech from any text using your cloned voice via MLX.
4
afplay
Audio plays back immediately. Or use --no-play to save only.
5
batch-tts.py
Process entire documents into audio clips with optional concatenation.
Key Features
  • Clone voice from a 3-second sample
  • ~100s audio per generation (configurable via --max-tokens)
  • Batch documents to audiobooks with --concat
  • SSH remote TTS from any machine
  • 3 TTS engines: Qwen3, Kokoro, Chatterbox
  • Speed control and format options
  • MimikaStudio GUI + REST API
  • 100% offline, no data leaves the machine
